Завдання 1. Застосування алгоритму максимального потоку для логістики товарів

Реалізовано модель логістичної мережі постачання товарів від терміналів до магазинів
з використанням алгоритму **Едмондса–Карпа** для знаходження максимального потоку.

Опис реалізації
- Побудовано граф з терміналами, складами та магазинами відповідно до умови.
- Додано технічні вершини `SuperSource` та `SuperSink`.
- Алгоритм Edmonds–Karp реалізовано через BFS у залишковій мережі.
- Для кожного знайденого augmenting path фіксується фактичний потік
  між терміналом та магазином.
- Виводиться таблиця потоків **Terminal → Shop** та аналітичні висновки.
